##### 在 Windows 上，可以使用 `nvm-windows` 这个专门为 Windows 设计的 Node 版本管理工具。您可以从它的 GitHub 仓库下载安装程序：https://github.com/coreybutler/nvm-windows/releases

**安装指定版本的 Node.js：**

```
nvm install <version>
```

**切换到已安装的 Node.js 版本：**

```
nvm use <version>
```

**列出所有已安装的 Node.js 版本：**

```
nvm list
```

**列出可用的 Node.js 版本：**

```
nvm ls-remote
```
