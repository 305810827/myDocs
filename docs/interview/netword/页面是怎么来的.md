页面是怎么来的，首先是通过我们在浏览器输入的网址，这个网址会在浏览器的解析下找到对应的服务器，向服务器请求返回一个html文件（其他资源不谈），html内容就是开发者写的代码，浏览器拿到html内容字符流，通过状态机去解析出里面的词（token），相当于浏览器可拆分识别的最小单位。

```
<p class="a">text text text</p>
```

这段代码拆分词（token）有

```
<p              “标签开始”的开始;
class="a"       属性；
>               “标签开始”的结束;
text text text  文本; 
</p>            标签结束。
```
